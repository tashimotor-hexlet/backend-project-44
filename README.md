# Brain Games

Brain Games — набор консольных мини-игр для тренировки логики и арифметики.

## Минимальные требования

- Node.js >= 16.x
- npm >= 7.x

## Установка

```sh
git clone https://github.com/tashimotor-hexlet/backend-project-44.git
cd backend-project-44
npm install
```

## Запуск игр

Установите пакет глобально:

```sh
npm link
```

Запустите любую игру командой:

- `brain-games`
- `brain-even`
- `brain-calc`
- `brain-gcd`
- `brain-progression`
- `brain-prime`
